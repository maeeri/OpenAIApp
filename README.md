# Image editing and generating (and drawing) app

    A work in progress

    Generate images utilising the OpenAI API 
    Documentation: https://beta.openai.com/docs/introduction
    
    Upload images to MongoDb
    Edit and download images
    
    Draw on canvas
    Erase from canvas
